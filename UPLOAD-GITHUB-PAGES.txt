AUTOVIP WEB — SYSTEM DASHBOARD + MÈO NỀN TRONG SUỐT v1.4.4

1. Giải nén file ZIP.
2. Tải TẤT CẢ file bên trong ZIP lên đúng thư mục /autovip/ đang chứa index.html trên GitHub Pages.
3. Không tải cả thư mục cha lồng thêm một cấp.
4. Ghi đè toàn bộ file cũ bằng toàn bộ file trong gói v1.4.4.
5. Đảm bảo dashboard-v1.4.0.css, dashboard-v1.4.0.js, cat-typing-transparent.webp, incokit-workflow-v1.0.5.css và incokit-ui-v1.0.5.js nằm cùng cấp với index.html.
6. Cài hoặc Reload extension Autovip Full v2.2.0 tại chrome://extensions.
7. Đóng tab Web Autovip cũ, mở lại:
   https://tranquytruong0362683566-gif.github.io/autovip/?v=20260721-145-clean-alpha
8. Nhấn Ctrl + Shift + R một lần.

Web không còn ô Extension ID. Extension tự liên kết bằng content bridge chỉ trên đúng URL /autovip/.
