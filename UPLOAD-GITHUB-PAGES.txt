AUTOVIP WEB — GIAO DIỆN INCOKIT + TỰ LIÊN KẾT EXTENSION v1.0.5

1. Giải nén file ZIP.
2. Tải TẤT CẢ file bên trong ZIP lên đúng thư mục /autovip/ đang chứa index.html trên GitHub Pages.
3. Không tải cả thư mục cha lồng thêm một cấp.
4. Ghi đè toàn bộ file cũ bằng toàn bộ file trong gói v1.0.5.
5. Đảm bảo incokit-workflow-v1.0.5.css và incokit-ui-v1.0.5.js nằm cùng cấp với index.html.
6. Cài hoặc Reload extension v1.9.9 tại chrome://extensions.
7. Đóng tab Web Autovip cũ, mở lại:
   https://tranquytruong0362683566-gif.github.io/autovip/?v=20260720-105
8. Nhấn Ctrl + Shift + R một lần.

Web không còn ô Extension ID. Extension tự liên kết bằng content bridge chỉ trên đúng URL /autovip/.

THAY ĐỔI v1.0.5:
- Giao diện xanh đen theo phong cách Incokit, có nền lưới tương tác theo chuột.

- Gộp hai kho mẫu vào một khu vực chung.
