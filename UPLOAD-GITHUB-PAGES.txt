AUTOVIP — DARK GREEN GRID v1.0.3

1. Giải nén file ZIP.
2. Tải TẤT CẢ file bên trong ZIP lên đúng thư mục đang chứa index.html của GitHub Pages.
3. Không tải cả một thư mục cha lồng vào bên trong /auto/.
4. Xác nhận index.html và incokit-dark-v1.0.3.css nằm cùng cấp.
5. Mở trang với ?v=20260720-103 rồi nhấn Ctrl + Shift + R.

File CSS đang được gọi:
./incokit-dark-v1.0.3.css?v=20260720-103
