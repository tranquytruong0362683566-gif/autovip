AUTOVIP WEB — GIAO DIỆN INCOKIT + QUY TRÌNH AUTOGPT ĐẦY ĐỦ v1.1.0

1. Giải nén file ZIP.
2. Tải TẤT CẢ file bên trong ZIP lên đúng thư mục /autovip/ đang chứa index.html trên GitHub Pages.
3. Không tải cả thư mục cha lồng thêm một cấp.
4. Ghi đè toàn bộ file cũ bằng toàn bộ file trong gói v1.1.0.
5. Đảm bảo incokit-workflow-v1.0.5.css và incokit-ui-v1.0.5.js nằm cùng cấp với index.html.
6. Cài hoặc Reload extension Autovip Full v2.1.0 tại chrome://extensions.
7. Đóng tab Web Autovip cũ, mở lại:
   https://tranquytruong0362683566-gif.github.io/autovip/?v=20260720-110
8. Nhấn Ctrl + Shift + R một lần.

Web không còn ô Extension ID. Extension tự liên kết bằng content bridge chỉ trên đúng URL /autovip/.

THAY ĐỔI v1.0.5:
- Giao diện xanh đen theo phong cách Incokit, có nền lưới tương tác theo chuột.
- Sắp xếp lại chức năng theo đúng trình tự: nguồn nhóm → hàng đợi → soạn bình luận → kết quả.
- Gộp hai kho mẫu vào một khu vực chung.
- Tách riêng link bình luận thành công và link bị AI loại bỏ.
- Cả hai danh sách vẫn được dùng để lọc trùng ở lần quét tiếp theo.

THAY ĐỔI v1.1.0:
- Giữ nguyên toàn bộ giao diện Incokit v1.0.5.
- Ghép lại quy trình autogpt-main: quét link nhóm → đọc description → AI phân loại/tạo bình luận → gửi → theo dõi spam → đóng tab → lưu kết quả → nghỉ → lặp vòng.
- Khi có Apify token, web ưu tiên lấy URL qua Apify; khi chưa có token, tự chuyển sang bộ quét nhóm trực tiếp trong Extension.
- Khôi phục đầy đủ action quét nhóm và đọc bài của Extension.
- Đồng bộ UID, đăng xuất và đăng nhập Cookie trực tiếp giữa web và Extension.
